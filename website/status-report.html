Database

9 tables: users, test_results, vaccine_records, immunity_metrics, autoimmune_markers, allergen_data, neuroprotective_markers, immunity_passports, antibody_trends

API Endpoints

14 routes: dashboard, vaccinations, recommendations, trends, allergens, autoimmune, neuroprotective, antibody trends, passports, risk score, health, + 2 POST

Auth

AWS Cognito JWT tokens. Mobile uses REST API (no SDK dependency). Web uses amazon-cognito-identity-js. Both pass Bearer token to Lambda.
